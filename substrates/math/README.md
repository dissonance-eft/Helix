# Helix Math Substrate

**Status: Planned / Stub**

The Math substrate will provide invariant discovery and analysis over mathematical datasets — symbolic expressions, proofs, conserved quantities, and structural patterns.

## Planned Pipeline Stages

| Stage | Name | Description |
|-------|------|-------------|
| 1 | `dataset_ingestion` | Scan and ingest mathematical datasets |
| 2 | `symbolic_extraction` | Extract symbolic expressions and structures |
| 3 | `invariant_detection` | Detect invariants and conserved quantities |
| 4 | `proof_analysis` | Formal proof structure analysis |
| 5 | `knowledge_graph` | Entity registry + graph integration |
| 6 | `llm_interpretation` | LLM-assisted interpretation |

from substrates.math.pipeline import MathSubstratePipeline
